My Portfolio

Here are some of the things I've built:

  • Readefine

    • Readefine is a browser extension (built in JavaScript, HTML, CSS) that simplifies language across the internet. I designed and built all aspects of Readefine including the client extension, the database, the Readefine server, and the landing page.

    Check out a demo here:

  • Numu-Know (source)

    • Numu-Know is a responsive web app that renders a database of Paiute (a Native American language) texts. Numu-Know allows anyone to see community documents and, when signed in, create and manage their own documents. Numu-Know is designed to help Paiute learners quickly access additional information about a word by clicking on the word, which shows word attributes such as part of speech or gloss.

  • Language Explorer (source)

    • Language Explorer is a Streamlit (which is a Python visualizer similar to Plotly's Dash) app deployed on Heroku that allows you to create and view texts with attributes. It comes preloaded with the Greek New Testament, which contains tag, lemma, and gloss attributes. These attributes can be viewed by hovering over a word.

  • NLP Calculator (source)

    • NLP Calculator is a calculator web app that allows you to make natural language math calculations such as "What is 17 times 45 + 30 - 6 to the power of three?" NLP Calculator parses the question and calculates the answer (using the standard order of operations).
